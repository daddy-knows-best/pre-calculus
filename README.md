# pre-calculus

It's kind of late, but I would like to try to build some repos related to math.

For example, how to eliminate parameter t:


```math
\begin{eqnarray*}
x = \sqrt{t},\ y = \sqrt{16-t}\\
\\
t = x^2\\
y = \sqrt{16-x^2}\\
y^2 = 16-x^2\\
x^2 + y^2 = 16\\
x^2 + y^2 = 4^2\\
\end{eqnarray*}
```
